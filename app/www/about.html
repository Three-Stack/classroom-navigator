CPP Navigator

About

CPPNavigator is a tool designed to help find classrooms in Cal Poly Pomona. It was created by three students as a final project for the Software Engineering class.

As students at Cal Poly Pomona, we noticed that room numbers in buildings were sometimes confusing and unintuitive. This makes it difficult to find your classroom as the start of a new semester without being late for class. After asking around and getting stories from other students, we found that this was a common issue experienced at Cal Poly Pomona.

Our goal is to help navigate the buildings at Cal Poly Pomona, so we built a search tool to search for your class. You can select a class by section and find its building and room number. Then, you can explore a floor plan of the building to find exactly where the room is located.

Our navigation tool to display directions is only supported for Buildings 8 and 9, but our class search tool covers every class at Cal Poly Pomona.

This website is unofficial, and we are not associated with Cal Poly Pomona in any way. Cal Poly Pomona does not endorse, recognize, or assume accountability for any content on this website. Any information presented on this website may be outdated or incorrect, since class information can change without being reflected on our website.

How does it work?

We get class information for each semester from public search functions. If you've ever used Cal Poly Pomona's class search, you've probably found it to be incredibly slow and frustrating.

Luckily, we do all the class searching and processing for you. You can use our search tool to search for and get information about Cal Poly Pomona classes, even if you're not looking for directions. Our search tool is much faster, so you can search the Cal Poly Pomona class databse quickly and easily.

To display directions, we use floor plans of each building. Unfortunately, these floor plans are not publicly available. We were only able to get floor plans for Buildings 8 and 9 for now.

This means that our search tool covers all classes at Cal Poly Pomona, but our direction tool only works for classes in Buildings 8 and 9. Once we obtain floor plans for more buildings, this tool may be updated in the future.

How to get started

To get started, click the Search button on the navigation bar to go to our search tool. You can either search for classes, specificl classrooms, or look at the building maps to review the floor plans.

To use the search tool, input any criteria into any of the search boxes. Our search tool will return a list of all classes matching your search. You can then view your search results to find your exact class. If your class is in a building supported by our navigation tool, you can click "Get Directions" to view exact directions to your class.

To search for specific classrooms, you can change the tab on the home page to the room search tab and enter the building and room numbers. Searching will display the floor plans and the directions to the room.

Finally you can view the floor plans for the buildings by changing the tab to building maps and selecting the building a floor that you want to view.

Our search function matches all fields which you entered. A blank field acts as a wildcard (matches anything for that field).

Search examples:
Course ID "CS 4800" will find all sections of the course CS 4800 - Software Engineering.
Course ID "CS 4800" and Section Number "01" will find the exact class CS 4800.01 - Software Engineering.
Only searching by Section Number "01" will find all classes with section 01. This is not very useful on its own.
Searching for Course ID "CS" will return any class containing CS - this returns CS classes, but also ECS classes such as ECS 3800.
You can also search by Instructor - searching for "Sun" will return all classes taught by Yu Sun but also other professors such as Peng Sun and Sunny Lie.
Using a combination of these search fields is best for finding the exact class you're looking for.

How to use the navigation tool

Clicking on the "Get Directions" button for a search result will bring you to the floor plan for the building which the class is in. You will be brought to the ground level of the building.

If the classroom is on the ground floor, you will see an arrow pointing towards the class from the building's entrance. Keep in mind that buildings have multiple entrances, so our navigation tool assumes you use the most commonly used entrance.

If the classroom is on a different floor, you will see an arrow pointing towards either the elevator or the stairs. Then, you can click "Class Floor" to see the map for the floor which the class is on. An arrow will guide you to your class from the stairs or the elevator to the classroom.
